Добавлена фича от Arigato -- подсветка размеров статических объектов карты
Добавлено маленькое баловство -- в меню установки/редактировании юнита,
при выборе человека анимированно гаснет и disable'ится пункт "топливо",


В настройках можно отключить автофокус и альтернативную подсказку на карте.
В файле профиля "Путь до вашего армовского профайла\scripts\XGUI-ExtendedGUI\settings" поместить переменные:

    // отключает автополучение фокуса контролом при наведении мыши
    _disableAutoFocus = true;
    // отключает подсказки по объектам отображаемые в режиме показа id
    _disableMapHint = true;
    // отключает подсвечивание boundingbox работающий в режиме показа id
    _disableBBHighlighting = true;

Например на моей машине этот файл лежит тут:
"C:\Documents and Settings\WinUser\Мои документы\ArmA 2 Other Profiles\Vdmj\scripts\XGUI-ExtendedGUI\settings"
